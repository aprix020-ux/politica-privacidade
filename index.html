Política de Privacidade

Esta Política de Privacidade explica como coletamos, usamos e protegemos os dados processados através deste aplicativo e integrações externas como Facebook Ads, WhatsApp API e ferramentas de automação.

Coleta de Dados

Coletamos apenas dados necessários para realizar automações solicitadas pelo usuário, como IDs de contas, tokens de acesso ou informações técnicas.

Uso das Informações

As informações são utilizadas exclusivamente para processar automações e integrações. Não vendemos, compartilhamos ou comercializamos informações pessoais.

Armazenamento e Segurança

Os dados podem ser armazenados temporariamente em servidores utilizados para processamento. Mantemos medidas de segurança técnicas e administrativas.
